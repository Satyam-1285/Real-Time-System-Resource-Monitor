# Real-Time-System-Resource-Monitor# 🚀 Terminal System Pulse

A lightweight, real-time system resource monitor built with Python. 

## ✨ Features
- **Live CPU Tracking**: Visualized with dynamic progress bars.
- **Memory Insights**: Real-time RAM usage monitoring.
- **Cross-Platform**: Works on Windows, macOS, and Linux.

## 🛠️ Installation
```bash
pip install -r requirements.txt
python src/monitor.py
